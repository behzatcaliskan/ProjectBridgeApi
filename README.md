# Project Bridge API

Bu proje, ASP.NET Core Web API kullanılarak geliştirilmiş bir backend servis uygulamasıdır. Entity Framework Core ile veritabanı bağlantısı kurulmuş ve API endpointleri controller yapısı üzerinden hazırlanmıştır.

## Kullanılan Teknolojiler

- ASP.NET Core Web API
- C#
- Entity Framework Core
- SQL Server
- REST API
- Migration
- Swagger / HTTP test yapısı

## Proje Özellikleri

- RESTful API mimarisi
- Controller tabanlı endpoint yapısı
- Model sınıfları
- Entity Framework Core ile veritabanı işlemleri
- Migration kullanımı
- SQL Server bağlantısı
- Backend servis geliştirme pratiği

## Proje Amacı

Bu proje, ASP.NET Core Web API mantığını öğrenmek ve frontend uygulamalarına veri sağlayabilecek backend servis yapısını geliştirmek amacıyla hazırlanmıştır.

## Teknik Kazanımlar

- API endpoint oluşturma
- HTTP GET, POST, PUT ve DELETE mantığı
- Entity Framework Core ile veri yönetimi
- Backend servis yapısının MVC projelerinden ayrıştırılması
- API ve veritabanı bağlantısının uygulanması
